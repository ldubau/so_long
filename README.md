# Description

So_long project is a 2D game were the goal il to collect all the collectibles then reach the exit case,<br>
the map has to have some minimal condition :
- be a square
- be surronded by wall ('1')
- have at least one collectible, an exit and a player start pos ('C', 'E', 'P')
- the map can be complite
If one of the previous conditions wasn't respected, a focus error message will apear and the programme will shut down.<br>
Else the game will start and you will be able to play with the (W, S, A, D) keycases, the numbers of mooves ou did will dynamicly printed into your terminal.

# Instructions

To run the project write ```make``` in your terminal and ```./so_long ./maps/<map name>```
replace "map name" with the name of the map of your choice.<br>
The game will close automaticly when you finish it, else you can click on the cross or tap the ESC keycase depend on your choice.
